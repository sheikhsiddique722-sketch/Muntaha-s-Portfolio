# Muntaha Shafique — Portfolio

Personal portfolio website showcasing my work as a Frontend Developer 
and Software Engineering student based in Karachi, Pakistan.

## 🔗 Live Demo 
https://sheikhsiddique722-sketch.github.io/muntaha-shafique-portfolio/

## ✨ Features
- Dark / Light theme toggle
- 3D animated background (canvas) 
- Aurora glow + animated blobs.
- Glassmorphism cards
- Smooth scroll (Lenis)
- GSAP animations
- Contact form with Firebase database + EmailJS notifications
- Fully responsive design
- CV download

## 🛠️ Tech Stack
HTML · CSS · JavaScript · GSAP · Firebase · EmailJS
